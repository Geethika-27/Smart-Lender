# 🏦 Smart Lender – Loan Eligibility Prediction System

Smart Lender is a Machine Learning-powered web application that predicts whether a loan applicant is eligible for approval based on various financial and personal attributes. The application provides real-time predictions, confidence scores, and risk categorization to assist in faster and more informed loan decisions.

---

## 🚀 Features

- 🤖 Loan Eligibility Prediction using XGBoost
- 📊 Prediction Confidence Score
- ⚠️ Risk Band Classification (Low, Moderate, High)
- 📱 Responsive Glassmorphism User Interface
- ✅ Input Validation
- 📈 Estimated Loan Burden Calculator
- 🌙 Light/Dark Theme Support
- 💾 Local Draft Saving
- ⚡ Real-Time Predictions

---

## 🛠️ Technology Stack

- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** Flask (Python)
- **Machine Learning:** XGBoost, Scikit-learn
- **Data Processing:** Pandas, NumPy
- **Deployment:** Render
- **Version Control:** Git & GitHub

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/smart-lender.git
```

2. Navigate to the project directory

```bash
cd smart-lender
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python Flask/app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
